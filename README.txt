# Lawrence Weather Plugin

**Contributors:** Lawrence Chibonfo  
**Tags:** weather, OpenWeather, shortcode, Elementor  

A custom plugin to fetch and display weather data from the OpenWeather API.

## Description
The Lawrence Weather Plugin retrieves weather data using the OpenWeather API and displays it on your site. You can use shortcodes to show the weather for specific cities and style the display with Elementor.

## Installation
1. Upload the plugin files to the `/wp-content/plugins/lawrence-weather-plugin` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress.
3. Navigate to **Settings → Weather Plugin** to enter your OpenWeather API key.

## Usage
- Add `[lawrence_weather city="London"]` to any page to display the weather.
- Use `[lawrence_weather_with_dropdown]` for a dropdown allowing users to select a city.

## Frequently Asked Questions
### How do I get an API key?
Visit [OpenWeather](https://openweathermap.org/) and sign up for a free API key.

### Where do I enter my API key?
Go to **Settings → Weather Plugin** to enter your OpenWeather API key.
